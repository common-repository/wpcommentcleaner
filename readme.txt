== Description ==

WPCommentCleaner is a very simple tool which can batch process SPAM, un-apporved and approved comments.
I have developed WPCommentCleaner for personal use.
Just thought it may come handy to someone else, hence I submitted this plugin to wordpress.org plugin repository.

== Installation ==

1. Upload `WPCommentCleaner` folder to the `/wp-content/plugins/` directory
1. Activate WPCommentCleaner through the 'Plugins' menu in WordPress
1. Go To The Plugin Admin Page Under WPCommentCleaner Menu
